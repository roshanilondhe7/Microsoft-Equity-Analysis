# Microsoft Stock Performance Analysis Dashboard

## Overview

This project presents a financial analysis of Microsoft stock using historical market data from **December 2020 to November 2021**.
The objective was to evaluate Microsoft's **performance, risk, and risk-adjusted returns** using financial metrics and visual analytics.

An interactive dashboard was developed in **Microsoft Excel** to illustrate key performance indicators and time-series trends.

---

## Objectives

* Analyse Microsoft's stock performance over time
* Measure **returns and volatility**
* Evaluate **risk-adjusted performance**
* Visualise trends through an analytical dashboard

---

## Tools Used

* Microsoft Excel
* Financial return calculations
* Time-series analysis
* Data visualisation

---

## Key Financial Metrics

| Metric                | Value  | Interpretation                                     |
| --------------------- | ------ | -------------------------------------------------- |
| Average Daily Return  | 0.18%  | Average daily growth in stock price                |
| Annualised Return     | 44.68% | Estimated yearly return based on daily performance |
| Annualised Volatility | 20.12% | Measures yearly risk or price fluctuation          |
| Sharpe Ratio          | 2.02   | Indicates strong risk-adjusted performance         |

---

## Dashboard Features

The dashboard includes:

* **Cumulative Return**

  * Shows total investment growth over time

* **Price vs 20-Day Moving Average**

  * Indicates momentum and trend direction

* **Monthly Average Closing Price**

  * Highlights price progression throughout the year

* **20-Day Rolling Volatility**

  * Shows how risk changes over time

* **Key Performance Indicators**

  * Annualised Return
  * Volatility
  * Sharpe Ratio
  * 52-Week High and Low
  * Average Trading Volume

---

## Key Insights

* Microsoft generated a **52.9% cumulative return** during the analysed period.
* The **annualised return of ~44.7%** indicates strong capital appreciation.
* Annualised volatility of **~20%** suggests moderate risk typical of large-cap technology stocks.
* A **Sharpe Ratio above 2** reflects excellent risk-adjusted performance.

Overall, the stock demonstrated **consistent upward momentum with efficient risk-adjusted returns**.
